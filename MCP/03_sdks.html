SDKs

원문: https://modelcontextprotocol.io/docs/sdk

MCP 서버와 클라이언트를 만들기 위한 공식 SDK 목록입니다. SDK는 기능 완성도·프로토콜 지원 범위·유지보수 약속 수준에 따라 티어로 분류됩니다. 자세한 기준은 SDK 티어 기준을 참고하세요.

Getting Started

각 SDK는 동일한 기능을 제공하되 언어 관용구와 모범 사례에 맞춰 설계되었습니다. 모든 SDK는 다음을 지원합니다.

  • 도구·리소스·프롬프트를 노출하는 MCP 서버 생성
  • 어떤 MCP 서버와도 연결 가능한 MCP 클라이언트 제작
  • 로컬/원격 전송(transport) 프로토콜 지원
  • 타입 안정성과 함께 프로토콜 준수

사용하려는 언어의 SDK 페이지에서 설치 방법, 문서, 예제를 확인하세요.
