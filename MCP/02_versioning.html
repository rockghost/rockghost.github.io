버전 관리

원문: https://modelcontextprotocol.io/specification/versioning

Model Context Protocol은 YYYY-MM-DD 형식의 문자열 버전 식별자를 사용합니다. 이 값은 호환성이 깨지는 변경이 마지막으로 발생한 날짜를 나타냅니다. 호환성을 유지하는 변경은 버전을 올리지 않고 점진적으로 반영됩니다.

리비전 상태

  • Draft: 진행 중인 사양으로, 아직 안정적으로 사용하기 어려움
  • Current: 현재 사용 가능한 사양. 하위 호환 변경이 계속 반영될 수 있음
  • Final: 완료된 과거 사양. 더 이상 변경되지 않음

현재 프로토콜 버전은 2025-11-25입니다.

버전 협상

버전 협상은 초기화 단계에서 수행됩니다. 클라이언트와 서버는 여러 버전을 동시에 지원할 수 있지만, 세션에는 단 하나의 버전에 합의해야 합니다. 호환되는 버전을 찾지 못하면 적절한 오류 처리를 통해 연결을 정리하도록 설계되어 있습니다.
