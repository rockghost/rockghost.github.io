원격 MCP 서버 연결하기

원문: https://modelcontextprotocol.io/docs/develop/connect-remote-servers

원격 MCP 서버는 인터넷에 호스팅된 도구, 서비스, 데이터 소스에 접근할 수 있게 해줍니다. 이를 통해 AI 어시스턴트를 단순한 도구가 아니라 실제 리소스에 접근하는 협업 파트너로 확장할 수 있습니다.

이 문서는 Claude의 Custom Connectors를 예로 들어 원격 MCP 서버를 연결하는 방법을 설명합니다. 개념 자체는 다른 MCP 호환 클라이언트에도 동일하게 적용됩니다.

원격 MCP 서버 이해하기

원격 MCP 서버는 로컬 서버와 유사하게 도구·프롬프트·리소스를 제공하지만 인터넷 상에 호스팅됩니다. 프로젝트 관리 도구, 문서 시스템, 코드 저장소 등 다양한 API 서비스와 연결할 수 있으며, 설치 없이 어디서든 접근할 수 있다는 점이 핵심 장점입니다.

Custom Connectors란?

Custom Connector는 Claude와 원격 MCP 서버를 연결하는 다리 역할을 합니다. 이를 통해 Claude가 외부 도구의 맥락을 활용하여 더 정확하고 풍부한 작업을 수행할 수 있습니다.

  • 직접 원격 MCP 서버 구축

원격 MCP 서버 연결 절차

  1. Connector 설정 열기

    Claude를 열고 프로필 아이콘 → Settings → 좌측 Connectors로 이동합니다.

  2. Custom Connector 추가

    하단의 “Add custom connector” 버튼을 클릭합니다.

  3. 서버 URL 입력

    서버 개발자/관리자가 제공한 URL을 입력하고 “Add”를 누릅니다.

  4. 인증 완료

    대부분의 원격 서버는 OAuth, API 키, 사용자/비밀번호 등의 인증을 요구합니다.

  5. 리소스와 프롬프트 사용

    연결 후 입력창의 클립 아이콘을 눌러 리소스/프롬프트를 선택할 수 있습니다.

  6. 도구 권한 설정

    Connector 설정에서 도구별 사용 권한과 보안 옵션을 조정합니다.

원격 MCP 서버 활용 베스트 프랙티스

  • 신뢰할 수 있는 서버만 연결하고, 요청되는 권한을 꼼꼼히 검토합니다.
  • 여러 Connector를 사용할 경우 목적/프로젝트별로 정리하고 주기적으로 정리합니다.
